# Amsa Bhutan

Brand profile mockup site for **Amsa** — a small, intentional resort in the Khotokha Valley, Bhutan.

Built from rammed earth. Rooted in the land.

## Stack

Single-file HTML/CSS — no build step, no dependencies. Fonts loaded from Google Fonts. All images embedded as base64.

## Development

Open `index.html` directly in a browser, or serve locally:

```bash
npx serve .
```

## Deploy

This site is designed to deploy to GitHub Pages:

1. Push to the `main` branch of your repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://adventure-access.github.io/amsa-bhutan`
